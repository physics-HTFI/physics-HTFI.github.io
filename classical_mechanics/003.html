3. 壁との衝突

衝突直後のボールの速度を求める公式。
この章のシミュレーション：

弾性衝突による速度の変化を知りたい

右図のように、壁や床で囲まれた空間にボールを投げ入れることを考える。ボールは、重力を受けて運動し、壁面に衝突すると跳ね返る。これまでと同様、この時のボールの運動 x(t) を計算したい。考える必要があるのは、衝突時の運動の変化である（それ以外は第1章のキャッチボールの運動と同じ）。衝突直後のボールの位置と速度が分かればそれらを初期値として、その後の運動が決まるわけだが、位置は衝突の前後で変わらないので、結局、衝突直後の速度 ˙xout を（衝突前の速度 ˙xin と衝突位置 x から）求めればよい。
ただし、ボールの弾み方はボールや壁の素材によって異なり、よく跳ねるものもあれば、そうでないものもある。そのため、「弾みやすさ」を決めてやらないと ˙xout は決まらないはずである。そこで、最もよく弾む理想的な場合を考えることにする。これを弾性衝突という。
この章では、まず幾何学的な考察により、弾性衝突における ˙xout を求める一般的な公式(11)を導き（3.1節）、その後、今度は力学的に考えることで、同じ公式を再導出する（3.2節）。最後に、具体的な計算を行い、球の場合の公式が式(17)となることなどを見る（3.3節）。

3.1 幾何学的導出

この節では、幾何学的な考察により、衝突後の速度 ˙xout の公式(11)を導く。具体的には、速度 ˙x の「壁と垂直な方向の成分」が、衝突の前後で反転するという条件を課せばよい。

衝突後の速度 ˙xout は、「壁と垂直な方向の成分」が反転する：式(2)

平らな硬い床に、よく弾むボールを垂直に落としてみる。すると、経験的に分かるように、衝突の前後でボールの運動方向は反転するが、速さは変わらない。一方、斜めにぶつけた場合、右図のように、衝突後の速度 ˙xout のうち、「壁と平行な成分」は衝突前と変わらず、「床と垂直な成分」は反転する。（現実的には、平行成分も摩擦の影響で多少変化するが、ここでは無視できるとする。）
これを数式で一般的に表すためには、衝突前の速度 ˙xin を、壁に対して「平行な成分 ˙x∥in 」と「垂直な成分 ˙x⊥in 」に分解すればよい： ˙xin=˙x∥in+˙x⊥in そうすれば、衝突により垂直成分 ˙x⊥in が反転するのだから、 ˙xout は以下のようになる： ˙xout=˙x∥in−˙x⊥in(1)
各成分 ˙x∥in,˙x⊥in を求めたいわけだが、これは、「壁面への正射影行列 P∥ 」および「壁面に垂直な方向への正射影行列 P⊥ 」を用いて、それぞれ ˙x∥in=P∥˙xin,˙x⊥in=P⊥˙xin と書ける（以下の【3.1-注1】の式(3)）。これらを代入すれば、上式(1)は ˙xout=(P∥−P⊥)˙xin=(1−2ˆaˆaT)˙xin(2) となる。ただし、 ˆa は、壁に垂直な単位ベクトルである（同注の式(4)）。よって後は、衝突点での ˆa を求めることさえできれば、上式(2)が確定し、 ˙xout が計算できることになる。

壁を定義する拘束条件 G(x) ：式(5)

壁に垂直なベクトル a を求めたいのだが、もちろんこれを計算するには、壁の形状が予め与えられていることが必要である。そのために、ボールの位置 x が壁の外にある時に正の値を取り、壁の内部で負の値を取るような関数 G(x) を導入する（右図）。つまり、 x は、不等式 G(x)≥0(5) を満たす点のみを取ることができ、不等式が破れる点が壁になる。この不等式、または G を拘束条件と呼ぶ。
例えば、半径 R の球の内部にボールを閉じ込める場合、ボール位置 x の取りうる範囲は |x|≤R で表されるので、拘束条件は G(x)≡R−|x|≥0 とすればよい。なお、 G の取り方には任意性があり、例えば G≡R2−|x|2 などとしてもよい。（記号 ≡ は、定義または恒等式であることを表すもので、方程式でないことを強調する。例えば上式の左側は拘束条件 G の定義であり、 x に対する方程式ではない。ただし自明な場合には、定義や恒等式であっても = と書く。）

壁面に垂直なベクトル a ：式(7)

後は、拘束条件 G(x) から、壁に垂直なベクトル a を求めるには方法が分かればよい。まず、壁面の形状は G(x)=0 により決まり、衝突点 x はこれを満たす。よって幾何学的には、 G(x)=0 を満たす曲面の接平面を求めれば、それに垂直なベクトルとして、 a を求めることができるはずである。
接平面の方程式を求めるには1次近似を行えばよい。これは、1変数関数のグラフを1次近似したものが、接線になるのと同様である（第1章の【1.1-注1】）。壁面上の点 x から微小量 δx だけ離れた点 x+δx が壁面上に留まるための条件は、 G(x+δt)=0 である。これを、1次近似すると次のようになる：（以下の【3.1-注3】の式(8)と式(10)） 0=G(x+δx)≐0G+dGdxδx=(∇G)Tδx(6) これが、 x+δx が接平面上にあるための条件である。
この式(6)は、「内積が0」という形をしているので、 ∇G は、接平面に直交していることが分かる（右図）。よって、これを壁面に垂直なベクトル a として採用できる： a=∇G≡⎡⎢⎣∂xG∂yG∂zG⎤⎥⎦(7)

【3.1-注3】多変数関数の1次近似：式(8)

2変数 x=(x,y) の任意の関数 G(x) のグラフを考える。このグラフ上の任意の点 P の近傍を拡大していくと、右図のように次第に平面に近づいていく。これは、球面であるはずの地球表面が、地上の我々からは平らに見えることからも分かる。
平面の関数は一般に1次関数になるので、 G(x+δx) の場合も、着目している点 x の近傍において、 δx の1次関数で近似できることになる：（右辺は x での値） G(x+δx)≐G+dGdxδxdGdx=[∂G∂x∂G∂y](8)(9) 緑字部分は、 δx=0 の時に両辺が等しくなることから決まる。赤字部分がこうなることについては、例えば、式(8)において、 δx=[δx0] としてみればよい。これは、 y を固定して、 G を x の1変数関数とみなすことに対応する。よって、 δx の係数は、 ∂G∂x （＝ x による偏微分）でなければならない。 δy の係数についても同様である。
この議論は、より多変数の場合にも自然に拡張できる。例えば、3変数の場合、式(9)を以下で置き換えればよい： dGdx=[∂G∂x∂G∂y∂G∂z]

補足

式(8)の赤字部分の転置を、 ∇G と書く：（以下は3変数の場合） ∇G=(dGdx)T=⎡⎢⎣∂xG∂yG∂zG⎤⎥⎦(10) ∇ はナブラと読む。 ∂xG は ∂G∂x の略記である。
式(8)のように1次近似ができる時、 G は、その点 x において微分可能（あるいは全微分可能）であるという。1変数関数の場合と同様、角ばった点があったりすると、その点では微分できない。なお、 dGdx の各成分が存在しても（＝偏微分が存在しても）、 G が微分可能であるとは限らない。というのも、偏微分が存在していても x,y 方向に1次近似できることしか言えず、別の方向では1次近似できないかもしれないからである（そうそう見かけることはないだろうが）。

衝突後の速度 ˙xout ：式(11)

以上により、衝突後の速度 ˙xout の公式を得るには、公式(2)に、壁と垂直なベクトル a の式(7)を代入すればよい。4すると、以下のようになる：（ ˆ∇G は ∇G を正規化したもの： ˆ∇G=∇G/|∇G| ） ˙xout=[1−2ˆ∇G(ˆ∇G)T]˙xin(11) これが求めたかった公式であり、 ˙xout が計算できることになる。
なお、 ∇G は 0 になる可能性がある。例えば、任意の拘束条件 G に対し、 G′≡G(x)3 を考えてみる。 G′ は、壁の内部で負、外部で正となっているので、拘束条件として正しいが、壁面上で ∇G′=0 となる。こうなるのは、 G の値が壁面上で停留値をとるような場合である（第14章の14.2節参照）。こうなってしまうと、上式(11)が計算できないので、うまく G を取り直して、 ∇G≠0 が成り立つようにする。

3.2 力学的導出

既に目的の公式(11)は得られているわけだが、ここでは、力学的な視点から再度考察を行い、同じ結果を導く。

ダランベールの原理(12)と弾性衝突条件(13)

壁に衝突したボールが速度 ˙x の向きを変えるのは、拘束条件 G≥0 が破れるのを防ぐために働く拘束力 fc のためである（添え字 c は拘束を意味するconstraintの頭文字）。上述の通り、衝突の前後で変化するのは ˙x の「壁に垂直な成分」のみであるが、これは、 fc の向きが壁に垂直となることを意味する。即ち、 fc は以下のように書ける：（ λ は未定乗数と呼ばれる未知数） fc=λ∇G(12) この式をダランベールの原理と呼ぶ。未定乗数 λ は、衝突の瞬間にのみ 0 でない値を持つが、衝突の瞬間の短い間にも急激に変化するため、直接測定することは難しい。
ダランベールの原理(12)は、弾性衝突でなくても成り立つので、これとは別に、弾性衝突であるための条件があるはずである。それは、衝突の前後でボールの運動の勢いが減衰しないこと、即ち、速度の大きさが不変： |˙xout|=|˙xin|(13) であるという条件である。これを弾性衝突条件という。

弾性衝突の公式(11)の再導出

示したいのは、ダランベールの原理(12)と弾性衝突条件(13)から、弾性衝突の公式(11)が導かれることである。そのために、ダランベールの原理(12)を、衝突前後の速度 ˙xin,˙xout に対する関係式にしたい。ダランベールの原理により、拘束力の向きは壁に垂直なのだから、速度の変化も壁に垂直である。よって、以下の関係式が成り立つ：（ Λ は未知数） ˙xout=˙xin+Λ∇G(14)
後は、未知数 Λ を求めればよいわけだが、そのためには弾性衝突条件(13)（を簡単のため2乗したもの）： ˙xTout˙xout=˙xTin˙xin(15) を用いればよい。具体的には、式(14)を式(15)に代入して ˙xout を消去すればよい。すると Λ の2次方程式になり、その解は以下の2つである： Λ={0−2|∇G|−2(∇G)T˙xin Λ=0 の場合、速度が変化しないため、壁にめり込んで拘束条件を破ってしまう。よってこれは不適当なので、もう1つの解が正しい解である。これを式(14)に代入すると、 ˙xout が確定し、幾何学的に求めた式(11)に一致することが分かる。

3.3 計算

この節では、いくつかの例に対して、具体的な計算を行う。必要なのは、式(11)である：（再掲） ˙xout=[1−2ˆ∇G(ˆ∇G)T]˙xin(16) この式を用いて実際の計算を行うには、まず、衝突する瞬間、即ち G(x(t))=0 となる瞬間の時刻 t まで、前章の2.2節で述べた運動方程式 m¨x=f を用いて計算を進める（ f は例えば重力）。壁に接している状態で、上式を使って衝突後の速度 ˙xout を計算する。それ以降は、衝突位置と ˙xout を初期値として、再び衝突するまで m¨x=f を用いて計算を進めていけばよい。
なお、初期値の取り方によっては、壁に接触したまま、壁に沿うように運動する場合も考えられる。その場合は、衝突が一瞬で終わるという暗黙の仮定が成立しないので、これまでの議論が使えない。実際、式(16)をそのまま使うと、壁面と垂直な方向の速度成分は 0 なので、 ˙xout=˙xin となってしまい、壁が無いのと同じになる。この場合については、後の第6章で扱う、 G=0 という拘束条件を課している場合の議論を使うことになる。

これを用いて数値計算を行うと、右図のようになる。円・球の半径はともに R=1m である。

例えば、 f(x) として以下のもの：（ a,b,c は定数） f(x)=acosbx+cx2 をとることにして、数値計算を行うと右図のようになる。グリッドの間隔は 1m である。
ところで、衝突は計算誤差を増大させる性質を持つ。というのも、曲面に衝突した場合、位置のずれが運動方向のずれを生じさせるからである。そのため、すぐに誤差が無視できなくなり、計算が意味をなさなくなってしまう。実際、上のシミュレーションでも、（ステップ幅 δt が毎回異なる実装のため）時間が経つにつれて毎回全く異なる運動になる。結果が異なってしまうのは理論の欠陥のようにも見えるが、そうではなく、現実世界での再現困難性を反映しているのである。実際、曲がった床にボールを落として跳ねさせる実験を行うと、同じところから落としたつもりでも、数回跳ねると毎回違う運動になってしまう。この性質が、理論にも正しく反映されているわけである。
