16. 制約条件付き極値問題

制約されたグラフ上の極値問題。

制約条件付き極値問題

制約条件付き極値問題とは、例えば、右図の青曲線上での極大・極小を求める問題である。即ち、「元のグラフ（同図の場合赤曲面）を制限した領域」での極大・極小を求める、という問題である。同図の場合、 x の取りうる範囲を、単位円上に制限している（赤グラフと緑の円筒が交差する部分が青曲線）。（「極値問題」という代わりに、同じような文脈で、「最大・最小問題」や「最適化問題」と言ったりもする。）
この章では、まず、制約条件がない場合について、1変数関数の停留条件(1)および多変数関数の停留条件(2)を求める（それぞれ14.1節、14.2節）。その後、制約条件付き極値問題の停留条件(9)を導く（14.3節）。また、「制約条件付き」極値問題を「制約条件のない」極値問題に変換する手法である、ラグランジュの未定乗数法についても述べる（【14.3-注2】）。

16.1 1変数関数の極値問題

まず、制約がない場合の極値問題からはじめる。この節では、1変数関数の停留条件(1)を導く。

1変数関数の停留条件：式(1)

知りたいのは、右図のように、ある関数 f(x) が極大または極小となる点 x （＝極値点）である。同図からも分かるように、極値点では、グラフの接線の傾きが 0 になる。このように接線の傾きが0になる点のことを、停留点という。この「接線の傾きが 0 」という条件（＝停留条件）を、書き下したい。
接線を考えるということは、1次近似を考えることと同じである。任意の点 x の周辺で f(x+δx) を1次近似すると、 f の変化量 δf は以下のように書ける：（第1章の【1.1-注1】） δf≡f(x+δx)−f(x)≐dfdxδx 停留点条件は、「 δx を変化させても δf が変化しないこと」なので、赤字部分が 0 となることに等しい： dfdx=0(1)
この停留条件(1)を解けば、その解の中に全ての極値点が含まれることになる。ただし、停留条件は、あくまで接線の傾きが0になるための条件なので、その解が極値点になるとは限らない（以下の例題参照）。

例題

例として、右図の関数 f(x)=x5−x3 の極値点を求める。停留条件(1)は dfdx=5x2(x2−35)=0 となる。よって停留点は、この解となる3点 x=0,±√35 である。
同図より、 x=√35=0.775⋯ は極小点、 x=−√35 は極大点であり、 x=0 はどちらでもない。得られた停留点が実際に極値点になっているかどうかの判定は、グラフを見て行うこととし、深入りはしないことにする。

16.2 多変数関数の極値問題

この節では、停留条件(1)を多変数関数の場合に拡張し、式(2)を導く。まだ制約条件は考えない。

多変数関数の停留条件：式(2)

右図のような多変数関数 f(x) の極値点を知りたい。図からもわかるように、極値点では、グラフの接平面の傾きが 0 （＝水平）となっている。即ち、この場合も、接平面の傾きが 0 という停留条件を考えればよい。
1変数の場合と同様に、任意の点 x の周辺で f(x+δx) を1次近似すると、以下のようになる：（第3章の【3.1-注3】） δf≐(∇f)Tδx∇f≡[∂xf∂yf] よって、 δf が変化しないためには、赤字部分が 0 でなければならない： ∇f=0(2)
全ての極値点は、この停留条件(2)の解となる。逆に、この式を満たしたとしても、それが極値点であるとは限らない（以下の例題参照）。

例題

例として、右図の関数 f(x)=(x2−y2)e−x2−y2 の極値点を求める。停留条件(2)は ∇f=[∂xf∂yf]=2e−x2−y2[x(1−x2+y2)y(−1−x2+y2)]=0 となる。よって停留点は、この解となる5点 (x,y)=(±1,0),(0,±1),(0,0) である。
同図より、 (±1,0) は極大点、 (0,±1) は極小点であるが、 (0,0) はどちらでもない（馬の鞍のようなグラフになっているので鞍点という）。

16.3 制約条件付き極値問題

さて、ようやく制約条件を課した場合である。この節では、制約条件を課した時の停留条件が、これまでと同様の式(9)で与えられることを見る。また、制約条件がない場合の極値問題に変換する手法である、ラグランジュの未定乗数法（【14.3-注2】）についても述べる。

素直に書き下した停留条件：式(3)と式(5)→分かりづらい

右図は、冒頭の図と同じものである。赤グラフ f(x) の定義域を緑の円柱 G(x)≡√x2+y2−1=0 上に制約したもの（＝2つのグラフが交わる部分）が、青グラフである。
この青グラフ上での極値問題を考えたいわけである。同図からもわかるように、極値点ではやはり、青グラフの接線の傾きが0になっている。よって、制約条件がない場合と同様、「接線（より多変数の場合には接平面など）の傾きが0」という停留条件を考えればよいことになる： δf≐(∇f)Tδx=0(3) ただし、 x には制約条件 G=0 がかかっているので、 δx は任意の値を取ることができない。
よって考えるべきは、 δx が満たすべき条件である。今の例では制約条件は1つだけだが、より変数が多い場合には複数あってもよいので、制約条件をベクトル表記しておく： G(x)≡⎡⎢ ⎢⎣G1G2⋮⎤⎥ ⎥⎦=0(4) δx に対する条件は、点 x から δx だけ移動したときに、制約条件(4)が破れないこと、即ち δG≐(∇G)Tδx=0∇G≡[∇G1∇G2⋯]≡⎡⎢ ⎢⎣∂xG1∂xG2⋯∂yG1∂yG2⋯⋮⋮⋱⎤⎥ ⎥⎦(5) である。以上をまとめると、停留条件は、「式(5)を満たす全ての δx について、式(3)が満たされていること」となる。しかし、このままでは分かりづらい。

節空間の基底ベクトルが分かれば、停留条件は式(6)→しかし一般には分からない

もし、任意の点 x における制約条件の接空間（＝接線や接平面の総称、即ち、式(5)を満たす δx が張る空間）の基底ベクトルが分かれば、停留条件は簡単になる。これを見るには、まず、接空間上のあらゆる δx は、基底ベクトル a1,a2,… の線形和と1対1対応することに注目する： δx=a1δλ1+a2δλ2+⋯=[a1a2⋯]Aδλ これを式(3)に代入したものが任意の δλ で成り立つので、停留条件は以下のように簡単になる： (∇f)TA=0(6)
拘束条件が1つであれば、 A として、 ∇G1 に垂直なベクトルを取ればよいのですぐ求まる。とはいえ一般には、基底ベクトルの行列 A を求めるのは困難である。分かっているのは、 ∇G 、即ち、 a1,a2,… の全てと直交するベクトル ∇G1,∇G2,… である。どうすればよいだろうか。

正射影行列(8)を用いると、停留条件は式(9)

上記の議論を参考にするならば、 δx を、何らかの制約されていないベクトル δλ を用いて δx=Pδλ と書ければ、停留条件は (∇f)TP=0(7) となるわけである。このような都合の良い行列 P はないだろうか。
結論から言うと、 P として、接空間（右図の青色部分）への正射影行列を考えればよい。そうすると右図のように、任意のベクトル δλ に対し、 δx=Pδλ は制約条件 δG≐0 を満たす。逆に、節空間上の制約条件を満たすあらゆる δx は δx=Pδλ の形に書ける（ δλ=δx とするだけである）。
正射影行列 P は ∇G を用いて書き下すことができ、以下のようになる：（以下の【14.3-注1】の式(10)において、 A を ∇G としたもの） P=1−∇G[(∇G)T∇G]−1(∇G)T(8) 以上により、停留条件は、式(8)の正射影行列 P のもとで以下のようになる：（見やすくするために式(7)の転置を取った） P∇f=0(9)

【14.3-注1】正射影行列

複数の1次独立なベクトル a1,a2,… をまとめて、行列 A=[a1a2⋯] で表す。右図のように、 A が張る空間（＝ Aλ の形でかけるベクトルの集合が作る空間）を ⟨A⟩ とおき、 ⟨A⟩ の直交補空間（＝ ⟨A⟩ と直交するベクトルの集合が作る空間）と ⟨A⟩⊥ とおく。
この時、 ⟨A⟩ への正射影行列 P∥ および、 ⟨A⟩⊥ への正射影行列 P⊥ は、以下のようになる： P∥=A(ATA)−1ATP⊥=1−P∥(10)

補足

実際に、 P∥ が正射影行列になっていることを確かめてみる。そのためには、任意のベクトル x を、 ⟨A⟩ の要素 x∥ と ⟨A⟩⊥ の要素 x⊥ に分解： x=x∥+x⊥ した時に、 P∥x∥=x∥ かつ P∥x⊥=0 となることを言えばよい。まず、 x∥ については、 x∥=Aλ の形で表せることより P∥x∥=A(ATA)−1ATA1λ=Aλ=x∥ また、 x⊥ については P∥x⊥=A(ATA)−1ATx⊥0=0 よって、確かに成立している。

例題

例として、右図の関数 f(x)=x3(1−y) の極値点を、単位円型の制約条件 G1(x)≡|x|−1=0 のもとで求める。
まず、 G の微分は ∇G=∇G1=x|x|=[xy] なので、式(8)の射影行列 P は以下のようになる： P=[1−x2−xy−xy1−y2]=[−yx][−yx] これと、 f の微分 ∇f=x2[3(1−y)−x] を停留条件(9)に代入すると P∇f≡4(y+1)(y−1)2(y+14)[−yx]=0 となる（途中で制約条件 x2=1−y2 を使って、 x2 を消去した）。
よって停留点は、 y に関して y=±1,−14 を満たさなければならない。 x は制約条件から決まる。こうして得られる4点 x=(0,±1),(±√154,−14) が停留点である。上図より、 (−√154,−14) は極小点、 (√154,−14) は極大点であり、 (0,±1) はどちらでもない。
なお、ここでは感覚を掴むために式(9)を用いて計算したが、今の場合、式(6)を使ったほうが簡単である。実際、式(6)の A として、 ∇G1 と垂直なベクトル A=[−yx] を取ればよい。あるいは、極座標 (r,θ) を使うと θ に関する制約無し極値問題に帰着でき、より簡単に解ける。このように、制約条件を簡単に消去できないか考えることも有用である。

補足：ラグランジュの未定乗数法

上述の制約条件付き極値問題は、制約条件のない極値問題（14.2節）に変換できることが知られている。そのようにして何が嬉しいかというと、コンピュータを用いて数値的に問題を解く場合に、「制約条件の無い極値問題で使われる優れた計算手法」（準ニュートン法など）が、そのまま流用できるのである。（制約条件の有無にかかわらず、停留条件が解析的に解けることはあまりないので、多くの場合コンピュータを用いて数値的に解くことになる。）
方法は簡単で、以下の【14.3-注2】のようにすればよい。即ち、ラグランジュ関数(11)を定義し、制約のない停留条件(12)の解を求めれば、その時の x は、制約条件 G(x)=0 を満たしており、かつ、元の制約条件付き極値問題の停留点にもなっているのである。

【14.3-注2】ラグランジュの未定乗数法

まず、ラグランジュ関数と呼ばれる量 L を L(x,λ)=f+GTλ(11) と定義する。次に、 x,λ が自由な値をとれるとして、 L の停留条件を考える：（ ∇x は x での微分、 ∇λ は λ での微分） ∇L≡[∇xL∇λL]=0(12) するとこの解は、制約条件 G=0 のもとでの、停留条件(9)の解と一致する。
この手法をラグランジュの未定乗数法といい、 λ を未定乗数（あるいはラグランジュの未定乗数）という。

証明

停留条件(12)をあらわに書くと [∇xL∇λL]≡[∇f+(∇G)λG]=0(13) となる。第2成分は、制約条件 G=0 そのものである。従って、第1成分が停留条件(9)と等価であることを言えばよい。
まず、式(13)の第1成分に正射影行列 P （式(8)）を左乗すると、停留条件(9)に一致する。よって、式(13)⇒式(9)。逆に、停留条件(9)において λ=−[(∇G)T∇G]−1(∇G)T∇f とおけば、式(13)の第1成分に一致する。よって、式(9)⇒式(13)。 ■
