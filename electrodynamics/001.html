1. 静電荷のクーロン力

電荷は電場を作る。その電場を測定すれば電荷分布が分かる。
この章のシミュレーション：（別ウィンドウで開く）

クーロン力が知りたい

力には、力学編で出てきた重力や拘束力以外に、電磁気的な力も存在する。例えば、服で擦った下敷きは静電気を帯び、紙片を吸い付ける。この時に働いている力をクーロン力という（第3章で見るように、もう1つローレンツ力と呼ばれるものがある）。
電磁力学では、そのような力を受ける物体の運動 x(t) を計算することが目的となる。 x(t) 、は運動方程式 m¨x=f を解くことによって得られるのだから、電磁気力 f を求めることが主な目的となる。通常は電磁気学と呼ばれるのだが、あえて電磁「力学」と書いているのは、物体の運動のみを考えており、電気回路や誘電体などは扱わないからである。
この章では、上述のクーロン力を、時間変化しないような場合について扱う。まず、電荷を蓄える方法について述べる（1.1節）。その後、電荷の周りのクーロン力を与えるクーロンの法則(2)を与え（1.2節）、クーロン力の源となる電荷を定量化する（1.3節）。最後に、いくつかの電荷分布（直線・平面・球）の場合についてクーロン力の計算を行う（1.4節）。

1.1 電荷を溜める：ヴァンデグラフ起電機

典型的なクーロン力は、上述のように服で擦った下敷きなのだが、それでは理論的に扱いづらいので、まず、静電気を溜める方法について述べる。

電荷にはプラスとマイナスがある

静電気を帯びることを「帯電する」といい、その静電気の量を電荷という（どのように電荷を定量化するかは1.2節で述べる）。電荷には2種類あり、同種の電荷を持つ物体同士は反発しあい、逆に、異種であれば引き合うことが知られている。これら2種類の電荷を便宜的に、プラスの電荷、マイナスの電荷と呼んで区別する。例えば、毛皮と塩化ビニールを擦り合わせたときに、毛皮が帯びる電荷がプラス、塩化ビニールがマイナスである。毛皮同士や塩化ビニール同士は反発し合い、毛皮と塩化ビニールは引き合う。

ファラデーのアイスペール実験

電荷を蓄える手段が欲しいのだが、そのために着目するのは、ファラデーのアイスペール実験（Faraday's ice pail experiment）と呼ばれる実験である。アイスペールとは、氷を入れて保つための（金属製の）卓上容器である。この実験によると、右図のように、金属球の内部に帯電した物体を触れさせると、その電荷が金属球に奪われる。
特にこの性質は、金属球側が帯電しているかどうかとは無関係である。金属球が帯電してくるにつれて、それ以上電荷を受け取らなくなりそうな気がするが、そうではないのである（もちろん限界はあるが）。

ヴァンデグラフ起電機

従って、帯電した物体をたくさん用意しておくなどし、それらの電荷を次々に金属球に移していけば、大量の電荷を金属球に蓄えることができる。このような装置を、ヴァンデグラフ起電機という。

1.2 点電荷のクーロン力

ヴァンデグラフ起電機の要領で、任意の量の電荷を蓄えることができると分かったので、これを用いて、電荷同士の間に働く力であるクーロン力の性質を探ることができる。この節では、クーロン力を与える法則である、クーロンの法則(2)について述べる。

点電荷同士に働く力は、逆2乗則(1)に従う

まず、帯電させた小さな物体 A を用意し、静止させる。そして、その近くに別の帯電させた小さな物体 B を様々な位置においてみて、 B がどのような力を受けるかを観測する。この観測用の物体 B のことを試験電荷と呼ぶ。これにより、どのような位置関係の時にどのような力が働くのかが分かる。ただし、物体の形状による影響や、相互作用によって電荷の偏りが生じるのを除くために、 A,B はなるべく小さな物体とし、あまり近づけ過ぎないようにする。このような理想化した電荷のことを、点電荷という。
そのような実験を行った結果、以下のことが知られている。即ち、原点に A を置いた時、位置 x にある試験電荷 B が受ける力 f(x) は、以下のような逆2乗則に従う： f(x)=αˆx|x|2(1) f(x) をベクトル場で表すと、右図（ α が負の場合）のように、金属球の中心から放射状の向きを持ち、大きさ |f| は中心からの距離の2乗に反比例する。 α は、 A,B の電荷に依存する係数である。また、 α の符号は、 A,B が同符号の電荷を持っていれば「+」（反発力）、異符号であれば「-」（引力）となる。
数値計算を行うと、式(1)のクーロン力を受ける物体の運動は、右図のようになる。

電荷の定量化→クーロンの法則：式(2)

式(1)の比例係数を決めたいのだが、これは点電荷がどれだけ帯電しているかに依存するはずなので、電荷の定量化と合わせて行う必要がある。
電荷は、クーロン力に比例するように定量化すればよいだろう（質量の定量化が重力に比例するようにできたのと同じことを期待している）。まず、基準となる適当な点電荷 A を用意して、その電荷を1クーロン（ 1C ）と定義する。これを用いて未知の点電荷 B の電荷を定量するには、試験電荷を用意して、 A と B の周りでのクーロン力を測定すればよい。例えば、試験電荷を 1m に近づけた時に、 B から試験電荷が受ける力が A の場合の 2 倍になるならば、 B は 2C の電荷を持つ。より詳しく言えば、 A，B に対する式(1)の係数を測定した結果がそれぞれ kA,kB であったとすると、 B の電荷 qB は、 qB=kA/kB[C] となるわけである。
基準となる 1C の電荷はをどうとるかには任意性があるが、次のようにとることになっている。即ち、同じ大きさの電荷を持つ2つの点電荷を 1m だけ離して置いた時に、両者の間に働くクーロン力の大きさが 8.988⋯×109N である時、これらの電荷の大きさをともに 1C と定義する。 1C というのは非常に大きな電荷であり、 1kg の物体が地上で受ける重力 9.8N と比べても、桁違いに大きなクーロン力を受けることが分かる。定義の数値が中途半端な上に非常に大きな値になっているのは、本来クーロンの定義は、次章で扱う電流を用いてなされるためである。定義の数値が厳密な値でないのもそのせいである。次章でもう一度言及する。
以上の定量化のもとで、原点に qA[C] の点電荷を置いた時に、点 x に置かれた電荷 q[C] をもつ点電荷が受けるクーロン力 f は、以下のようになる： f=keqqAˆx|x|2ke≡8.988⋯×109C−2⋅kg⋅m3⋅s−2(2) これをクーロンの法則といい、係数 ke をクーロン定数という。分子に電荷の積が現れているのは、作用反作用の法則により A が B から受ける力が fA=−f となるからである。作用反作用の法則が成り立っていなかったら、例えば棒の両端に A,B を取り付けた時、棒全体にかかる力 fA+fB が 0 でないことになり、棒が勝手に加速し直感的にも不自然である。

1.3 密度分布を持った電荷が作る電場

これまでは、点電荷を考えてきたが、一般には電荷はさまざまな分布を持っている。そのような場合には、物体を小さく分割し、多数の点電荷の集合とみなせば、これまでの議論が使える。この節では、まず、試験電荷によらない量である電場を定義し、その後、密度分布を持った電荷が作る電場が、式(6)で与えられることを見る。

複数の点電荷から受けるクーロン力：式(3)

式(2)は点電荷 qA を原点に置いた場合のものであったが、任意の位置 xA に置いた場合には、単純に変更移動した以下の形になる： f(x)=keqqAˆx−xA|x−xA|2
次に、点電荷が複数ある場合を考える。電荷が加法性を持つように定義できたことからもわかるように、複数の点電荷 q1,q2,… がある場合、各々の点電荷からのクーロン力 f1,f2,… を足し合わせたものが、全体のクーロン力 f になる： f(x)=f1+f2+⋯=∑ikeqqiˆx−xi|x−xi|2(3) これを重ね合わせの原理という。
なお、クーロン力の加法性は、上記の電荷の定量化とも相性がよい。実際、電荷が qA,qB である2つの点電荷を合体させると、クーロン力の加法性により、電荷 qA+qB の点電荷のように振る舞う。つまり、電荷自体も加法性を持つようになっているのである。これはちょうど、力学の第2章で質量を定量化する際、加法性を持たせることができたのと同じである。

電場の導入：式(4)

式(3)から分かるように、試験電荷が受けるクーロン力は、試験電荷の電荷 q に比例することになる。そのため、この q を括り出してしまって、対象としている電荷に関する量だけにするのがよい。これを電場と言い E と書く。
例えば、原点に置いた電荷 qA が作る電場は、式(2)から q を除いたものなので、以下のようになる： E(x)=keqAˆx|x|2(4) 複数の点電荷がある式(3)の場合は E(x)=∑iEi(x)=∑ikeqiˆx−xi|x−xi|2(5) である。ただし、 Ei は i 番目の点電荷（電荷が qi で位置が ri ）が作る電場である。
点 x にある試験電荷 q が受ける力を、 E(x) を用いて書くと f=qE(x) となる。 q は試験電荷に対する量、 E(x) はソース電荷に対する量、という形に分離しているわけである。

電荷密度 ρ(x) が作る電場(6)

電荷が連続的に分布している場合には、力学の15.3節のように、電荷を持った物体を非常に小さな体積要素に分割し、各体積要素からの寄与を足し合わせれば、区分求積によって計算することができる。要は、(5)に現れる和を積分に置き換えればよい：（ ∫x′[⋯] は ∫[⋯]d3x′ を略記したもの） E(x)=∫x′keρ(x′)ˆx−x′|x−x′|2(6) ただし、 ρ(x′) は電荷密度と呼ばれるもので、位置 x′ にある微小体積 δV に対し、その電荷 δq が δq≐ρδV で近似できるような量である。 ρ が最初に与えられていればよいが、そうでない場合には、実験によって、 ρ の分布を決定しなければならない。実際に電場から ρ を逆算することができるのだが、数学的な議論が必要になるので、第○○章で行う。
ところで、式(6)の被積分関数は、 x′=x において発散する。そのため、この点の周辺で区分求積は定義できないように見える。しかし直感的には、点 x にある電荷が、同じ位置にでの電場に影響するとは思えない。実際、数学的にも、発散する部分からの寄与は消えることが言える（【1.3-注1】）。よって結局、発散する部分をくりぬいた状態で積分を定義し、くりぬいた部分を小さくする極限を取ることで問題なく定義できる。

【1.3-注1】広義積分

原点を含む3次元領域 D 上での積分 ∫x∈D1|x|2 は、非積分関数が発散する点を持つため、そのままでは定義できない。そこで、原点を含む微小な領域 ϵ でくりぬいた領域 D−ϵ を考える。この D−ϵ 上での積分において、 ϵ を原点に向かってい縮めていく極限 ϵ→∅ をとる： I=limϵ→∅∫x∈D−ϵ1|x|2 するとこの積分は、（極限の取り方によらず）収束する。このように、通常の積分では定義できないが、極限をとることでうまく定義できる積分を、広義積分という。

証明

積分が定義できないのは原点付近だけなので、 D として半径 1 の球を考えれば十分である。 ϵ に完全に含まれる最大の球（原点に置かれているとする）の半径を rmin 、逆に ϵ を完全に含む最小の球の半径を rmax とおくと、以下の不等式が成り立つ： ∫rmax<|x|<11|x|2≤∫x∈D−ϵ1|x|2≤∫rmin<|x|<11|x|2(7) 両端の項は、極座標を用いれば具体的に計算できる。例えば最左辺は ∫rmax<|x|<11|x|2=4π∫1rmax1r2⋅r2dr=4π(1−rmax) となる。これを式(7)に代入すると 4π(1−rmax)≤∫x∈D−ϵ1|x|2≤4π(1−rmin) となるので、 ϵ→0 の極限、即ち、 rmin,rmax→0 を取れば I=4π となり、確かに収束している。 ■

1.4 例題：電場の計算

電荷密度 ρ(x) の例として3種類：(A)無限に伸びた直線電荷、(B)無限に広がった平面電荷、(C)球対称な電荷分布を考える。電場の分布はそれぞれ【1.4-注1】、【1.4-注2】、【1.4-注3】のようになる

【1.4-注1】(A)直線電荷の周りの電場

無限に伸びた直線電荷の周りの電場 E(x) は、以下のようになる： E(x)=2keρˆx⊥|x⊥| ただし、直線電荷は、原点を通り、線密度は定数 ρ[C⋅m−1] とする。 x⊥ は、 x を電荷直線と垂直な方向へ正射影したものである。導線と x の距離 |x⊥| に反比例することが分かる。

導出

【1.4-注2】(B)平面電荷の周りの電場

無限に広がった平面電荷がの周りの電場 E(x) は、以下のようになる： E(x)=2πkeρˆx⊥ ただし、平面電荷は、原点を通り、面密度は定数 ρ[C⋅m−2] とする。 x⊥ は、 x を電荷平面と垂直な方向へ正射影したものである。面をまたぐと方向が変わるが、それ以外では定数となる。これにより、一様な電場を作ることができる。

【1.4-注3】(C)球対称な電荷の周りの電場

原点の周りに球対称な電荷密度 ρ(x) がある場合の電場 E(x) は、以下のようになる： E(x)=keQˆx|x|2Q(x)=∫|x′|≤|x|ρ(x′)d3x′ Q(x) は、半径が |x| の球内の全電荷である。これを見ると、 Q(x) が原点に集中した時と同じになっている一方、半径が |x| 以上の部分にある電荷による寄与は打ち消しあって 0 になっている。この性質をニュートンの球殻定理(Newton's shell theorem)という。
