14. 制約付き極値問題

ここから残りの章は、これまでの内容に関連する数学的な話題について取り上げる。
この章では、制約条件がついた場合の極値問題を考える。例えば、右図の青線上で極大・極小となる点を求めたい。

14.1 1変数関数の極値問題

まず、制約がない場合の極値問題からはじめる。

1変数関数の停留条件は式(1)

右図のように、ある関数 f(x) が極大または極小となる点 x （＝極値点）を知りたい。図からもわかるように、極値点では、グラフの接線の傾きが 0 になる。このように接線の傾きが0になる点のことを、停留点という。そこで、停留点になるための条件（＝停留条件）を、式で表すことを考える。
任意の点 x の周辺で f(x+δx) を1次近似（第1章の【1.1-注1】）すると、その変化量 δf は以下のように書ける： δf≡f(x+δx)−f(x)≐dfdxδx 停留点であるための条件は、 δx を変化させても δf が変化しないこと、即ち、赤字部分が 0 となることである： dfdx=0(1)
この停留条件(1)を解けば、その解の中に全ての極値点が含まれることになる。ただし、停留条件は、あくまで接線の傾きが0になるための条件なので、その解が極値点になるとは限らない（以下の例題参照）。

例題

例として、 f(x)=x5−x3 の極値点を求める。停留条件(1)は dfdx≡5x2(x2−35)=0 となる。よって停留点は、この解となる3点 x=0,±√35 である。
右図より、 x=√35=0.775⋯ は極小点、 x=−√35 は極大点であり、 x=0 はどちらでもないことが分かる。得られた停留点が実際に極値点になっているかどうかの判定は、グラフを見て行うこととし、深入りはしないことにする。

14.2 多変数関数の極値問題

次に上記の内容を、多変数関数に拡張する。まだ制約条件は考えない。

多変数関数の停留条件は式(2)

右図のような多変数関数 f(x) の極値点を知りたい。右図からもわかるように、極値点では、グラフの接平面の傾きが 0 （＝水平）である。
1変数の場合と同様に、任意の点 x の周辺で f(x+δx) を1次近似（第3章の【3.1-注2】）すると、以下のようになる： δf≡f(x+δx)−f(x)≐dfdxδx よってこの場合も、停留条件は、赤字部分が 0 となることである。 ∇ を使って書くと以下のようになる（ ∇f=(df/dx)T ）： ∇f≡⎡⎢ ⎢⎣∂xf∂yf⋮⎤⎥ ⎥⎦=0(2)

例題

例として、 f(x)=(x2−y2)e−x2−y2 の極値点を求める。停留条件(2)は ∇f≡[∂xf∂yf]≡2e−x2−y2[x(1−x2+y2)y(−1−x2+y2)]=0 となる。よって停留点は、この解となる5点 x=(±1,0),(0,±1),(0,0) である。
右図より、 (±1,0) は極小点、 (0,±1) は極大点であり、 (0,0) はどちらでもない。

14.3 制約付き極値問題

ここから制約条件を考える。

青グラフ上の極値点を知りたい

右図の青グラフは、赤グラフ f(x) の定義域を緑の円柱 G1(x)≡√x2+y2−1=0 上に制約したもの（＝2つのグラフが交わる部分）である。
今の例では制約条件は G1 の1つだけだが、より変数が多い場合には複数あってもよいので、 G(x)≡⎡⎢ ⎢⎣G1G2⋮⎤⎥ ⎥⎦=0(3) のようにベクトル表記しておく。
知りたいのは、青グラフ上での極値点、即ち、制約条件 G(x)=0 を満たす範囲に x を制限した時の f(x) の極値点である。同図より停留条件は、青グラフの接線の傾きが0になることである。

停留条件は式(6)

この場合も、停留条件は δf≐dfdxδx=0(4) である。ただし、 δx は、制約条件(3)を1次近似の範囲で満たすもの、即ち δG≐dGdxδx=0(5) でなければならない。まとめると、停留条件は「制約条件(5)を満たす全ての δx について、式(4)が満たされていること」となる。これを見やすくしたい。
第3章でも見たように、制約条件は、射影行列（第3章の【3.1-注1】）と相性がよい。まず、制約条件(5)を満たす δx が張る空間への直交射影行列 P は、以下のように書ける： P=1−(dGdx)T[dGdx(dGdx)T]−1dGdx すると、 δx=Pδy で定義された δx は、(A)常に制約条件(5)を満たし、かつ、(B)制約条件(5)を満たすあらゆる値をとりうる。実際、(A)は、 δx=Pδy を制約条件(5)に代入すればすぐ分かる。また、(B)は、制約条件(5)を満たす全ての δx が、 Pδx=δx を満たすことから分かる。
よってこの δx=Pδy を用いれば、制約条件(5)を課す必要がなくなり、停留条件は、式(4)に δx=Pδy を代入した (df/dx)Pδy=0 だけでよくなる。 δy は任意の値をとれるので、結局、赤字部分が 0 になればよい。これを、 ∇ を用いて書くと、停留条件は最終的に以下のようになる： P∇f=0(6) ただし P≡1−∇G[(∇G)T∇G]−1(∇G)T∇G≡[∇G1∇G2⋯](7) である（ ∇G=(dG/dx)T ）。 P は、上述のとおり、制約条件の接空間への直交射影行列である。

例題

例として、右図のように、定義域を単位円 G1(x)≡√x2+y2−1=0 上に制約したときの、 f(x)=x3(1−y) の極値点を求める。
まず、 G の微分は ∇G=x|x|=[xy] なので、 P は以下のようになる： P=[1−x2−xy−xy1−y2]=[−yx][−yx] これと、 f の微分 ∇f=x2[3(1−y)−x] を用いると、停留条件(6)は P∇f≡4(y+1)(y−1)2(y+14)[−yx]=0 となる（途中で制約条件 x2=1−y2 を使って、 x2 を消去した）。
よって、停留点は y=±1,−14 を満たす点である。 x は制約条件から決まる。こうして得られる4点 x=(0,±1),(±√154,−14) が停留点である。上図より、 (−√154,−14) は極小点、 (√154,−14) は極大点であり、 (0,±1) はどちらでもない。なお、この問題の場合は、極座標 (r,θ) を使うと θ に関する制約無し極値問題に帰着でき、より簡単に解ける。このように、制約条件を簡単に消去できないかと考えることも有用である。

補足：ラグランジュの未定乗数法

上述の制約条件付き極値問題は、制約条件のない極値問題に変換できることが知られている。そのようにして何が嬉しいかというと、コンピュータを用いて近似的に問題を解く場合に、制約条件がない場合に知られている優れた計算手法が、そのまま流用できるのである。
方法は簡単である。まず、ラグランジュ関数と呼ばれる量 L を、以下のように定義する： L(x,λ)=f+GTλ 後は、 x,λ が自由な値をとれるとして停留条件 ∇L=0 を考えれば、これは停留条件(6)と等価になる。よって、 ∇L=0 を満たす点を求めれば、その時の x は、元の制約条件付き極値問題の停留点になっているのである（もちろん G(x)=0 も満たされている）。この手法をラグランジュの未定乗数法といい、 λ を未定乗数という。
【証明】停留条件 ∇L=0 をあらわに書くと、 x での微分を ∇x 、 λ での微分を ∇λ と書くことにして ∇L≡[∇xL∇λL]≡[∇f+(∇G)λG]=0(8) となる。第2成分は、制約条件 G=0 そのものである。第1成分については、式(7)の射影行列 P を左乗すると、 P∇f=0 となり停留条件(6)に一致する。逆に、停留条件(6)は、常に上式の第1成分の形に書くことができる。これは、上式の第1成分に λ=−[(∇G)T∇G]−1(∇G)T∇f を代入すれば、停留条件(6)に一致することから分かる。以上により、両者は等価である。即ち、ラグランジュの未定乗数法の解は、対応する制約条件付き極値問題の解でもあり、その逆も成り立つ。 ■
